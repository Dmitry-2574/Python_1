Emmit и его возможности

Простые команды

  1. ! - базовая структура документа
  2. h1{Текст} - Создание заголовка первого уровня
  3. h2{Текст}+p{Текст} - создание заголовка второго уровня
  4. ul>li*3{$.Элемет списка} - создание списка из трех элеменов. Будет присутствовать нумерация

Сложные команды

p.text{Привет} - создание абзаца с классом text и с текстом "Привет"

Привет

span.text>code{print("Hello,Word)} - создание спана с классом text и вложенным тегом code и текстом "print("Hello, World!")" (Hello, Word)

h2 {Вставка изображения в HTML}+img[src="1.jpg" alt="Фотография пользователя" - создания заголовка второго уровня и вставка изображения с атрибутами src и alt

3 Способа стилизовать HTML
